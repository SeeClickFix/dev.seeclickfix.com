The SeeClickFix API

This describes the resources that make up the official SeeClickFix API v2. If you have any problems or requests please contact us developers.

Current Version

The SeeClickFix API v2 is currently not available until summer 2013. A notice will be given closer to the actual date. For now use version 1.

We consider the “beta” API unchangeable. File a support issue if you have problems.

Expected Changes

These changes are not implemented, just planned for the next major API version.
